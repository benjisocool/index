Website Report

This project helped me learn the basics of HTML and CSS and gave me more confidence in building a simple website from scratch. At the beginning, I found it difficult to understand how pages link together and how a stylesheet connects to an HTML file. After practising, I became more comfortable with creating different pages, adding navigation links, and using CSS to improve the appearance of the website.

One of the main parts of my website was creating a clear structure across all pages. I used the same navigation menu on each page so the user can move around the site easily. I also made sure that the website had the required pages, including a homepage, projects page, contact page, video demonstration page, and report page. This helped me understand the importance of consistency in web design.

For the design, I kept the layout simple because I wanted the site to be easy to read and easy to use. I used a black navigation bar with white text because it is clear and stands out well. I used Arial as the font because it is simple, readable, and works well on most devices. I also added spacing around the content so the page does not look too crowded.

The projects page uses a grid layout, which helped me practise using CSS grid. This was useful because it allowed me to display project sections in a more organised way. I also used a media query in my CSS so the layout changes on smaller screens. This helped me understand responsive design and how websites should work on both desktop and mobile devices.

During development, I had some problems with file names, linking pages, and making sure the CSS was working properly. I had to fix issues such as naming the home page correctly as index.html and checking that all links matched the correct file names. Debugging these problems helped me understand how small errors can stop a website from working properly.

Overall, this assignment improved my understanding of web development, design, and testing. If I improved this site in the future, I would add real project images, stronger styling, and a better contact system using a server-side language.
